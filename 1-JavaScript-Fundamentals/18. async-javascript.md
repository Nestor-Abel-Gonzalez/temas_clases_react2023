## Async/Await (ES2017)
Async/Await es Sugar Syntax que nos permite estructurar funciones as√≠ncronas sin bloquear el hilo principal, pero se define de la misma forma que definir√≠amos una funci√≥n s√≠ncrona tradicional. Esta forma nos evita el encadenar promesas expl√≠citamente, pero el comportamiento por debajo es igual al que vimos anteriormente.

```js

async function fetchData(urlApi) {
  const request = await fetch(urlApi);
  const data = await request.json();

  return data;
}

async function getData() {
  try {
    const urls = await fetchData(API_URL);
    const characters = await fetchData(urls.characters);
    const episode = await fetchData(characters.results[0].episode[0]);

    console.log(characters.results[0]);
    console.log(characters.results[0].name);
    console.log(episode.name);
  } catch (error) {
    console.log(error.message);
  }
}

getData();
```
